# Yunus_BSG

Bu klasör, "Şebekeden Bağımsız Güneş Enerjili Elektrikli Araç Şarj İstasyonlarına Yönelik Siber Saldırıların Araştırılması" başlıklı çalışmaya dayalı bireysel dokümantasyonumu içermektedir.

## İçerik

- `Hakkımda.md`: Kısa kişisel tanıtım ve ilgi alanları.
- `Anomali_Senaryosu.md`: Off-grid güneş enerjili EA şarj istasyonu için siber saldırı/anomali senaryosunun açıklaması.
- `SWOT.md`: İlgili makale temelinde hazırlanmış SWOT analizi.
